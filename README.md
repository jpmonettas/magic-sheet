# magic-sheet

Create magic sheets to improve your Clojure[Script] repl experience.

## Prerequisites 

- Install JDK 8
- If you install OpenJDK make sure you install openjfx

## Building 

```bash
lein uberjar
```

## Usage

```bash
java -jar target/uberjar/magic-sheet-0.1.0-standalone.jar localhost:40338
```
